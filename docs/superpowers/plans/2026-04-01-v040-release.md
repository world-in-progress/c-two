# v0.4.0 Release Plan

> **Status: 🔄 IN PROGRESS**
> **Branch:** `phase4/cleanup` → PR to `world-in-progress/c-two:main`
> **Scope:** 合并 Phase 1-4 成果，清理技术债，发布 v0.4.0

## 背景

从 v0.3.0 以来，项目经历了 4 个阶段的 Rust 传输层下沉（Phase 1-4），77 个 commit 滞留在 `phase4/cleanup` 分支。核心成果：

- **Rust 传输层完全接管**：c2-wire, c2-ipc, c2-server, c2-relay, c2-http 五个 crate
- **Python 代码大幅瘦身**：删除 ~3800 行冗余 Python 传输代码
- **三级内存降级**：Buddy SHM → Dedicated SHM → File Spill
- **自由线程安全**：所有 PyO3 类型 frozen，Python 共享状态加锁

当前状态：4,557 行 Python + 12,926 行 Rust，532 个测试全部通过。
